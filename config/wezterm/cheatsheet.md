# WezTerm チートシート

Leader: `Ctrl + a`（2秒以内に次のキーを入力）

## タブ

| 操作             | キー                  |
| ---------------- | --------------------- |
| 新規タブ         | `Cmd + t`             |
| タブを閉じる     | `Cmd + Shift + W`     |
| 次のタブ         | `Ctrl + Tab`          |
| 前のタブ         | `Ctrl + Shift + Tab`  |
| タブ番号で切替   | `Cmd + 1〜9`          |
| タブを左に移動   | `Leader` → `{`        |
| タブを右に移動   | `Leader` → `}`        |

## ペイン

| 操作             | キー                  |
| ---------------- | --------------------- |
| 縦分割（左右）   | `Cmd + d`             |
| 横分割（上下）   | `Cmd + Shift + d`     |
| ペインを閉じる   | `Cmd + w`             |
| 前のペインへ     | `Cmd + [`              |
| 次のペインへ     | `Cmd + ]`              |
| 左に移動         | `Leader` → `h`        |
| 下に移動         | `Leader` → `j`        |
| 上に移動         | `Leader` → `k`        |
| 右に移動         | `Leader` → `l`        |
| ペイン選択       | `Ctrl + Shift + [`    |
| ズーム切替       | `Leader` → `z`        |
| サイズ調整モード | `Leader` → `s` → `h/j/k/l`（`Enter`で終了） |

## Workspace

| 操作             | キー                  |
| ---------------- | --------------------- |
| 切り替え         | `Leader` → `w`        |
| 新規作成         | `Leader` → `Shift+W`  |
| 名前変更         | `Leader` → `$`        |

## コピーモード（`Leader` → `[` で開始）

### 移動

| 操作             | キー       |
| ---------------- | ---------- |
| カーソル移動     | `h/j/k/l`  |
| 単語移動         | `w` / `b` / `e` |
| 行頭（内容）     | `^`        |
| 行末             | `$`        |
| 行頭（左端）     | `0`        |
| 先頭へ           | `g`        |
| 末尾へ           | `G`        |
| ジャンプ         | `f/t` / `F/T` |
| ジャンプ繰り返し | `;`        |

### スクロール

| 操作             | キー        |
| ---------------- | ----------- |
| Page Up          | `Ctrl + b`  |
| Page Down        | `Ctrl + f`  |
| 半ページ下       | `Ctrl + d`  |
| 半ページ上       | `Ctrl + u`  |

### 選択・コピー

| 操作             | キー        |
| ---------------- | ----------- |
| 文字選択         | `v`         |
| 行選択           | `V`         |
| ブロック選択     | `Ctrl + v`  |
| コピー           | `y`         |
| コピーして終了   | `Enter`     |
| モード終了       | `Esc` / `q` |

## その他

| 操作             | キー                 |
| ---------------- | -------------------- |
| コマンドパレット | `Cmd + p`            |
| フルスクリーン   | `Alt + Enter`        |
| コピー           | `Cmd + c`            |
| ペースト         | `Cmd + v`            |
| フォント拡大     | `Ctrl + +`           |
| フォント縮小     | `Ctrl + -`           |
| フォントリセット | `Ctrl + 0`           |
| 設定再読み込み   | `Ctrl + Shift + r`   |
