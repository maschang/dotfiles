# Neovim プラグイン チートシート

リーダーキー: `<Space>`

## Telescope（ファジーファインダー）

| キー | 操作 |
| --- | --- |
| `<Space>ff` | ファイル検索 |
| `<Space>fg` | 全文検索（ripgrep） |
| `<Space>fb` | バッファ一覧 |
| `<Space>fh` | ヘルプタグ検索 |

Telescope 起動後:

| キー | 操作 |
| --- | --- |
| `<C-n>` / `<C-p>` | 候補を上下移動 |
| `<CR>` | 選択して開く |
| `<C-x>` | 水平分割で開く |
| `<C-v>` | 垂直分割で開く |
| `<C-t>` | 新しいタブで開く |
| `<Esc>` | 閉じる |

## Oil（ファイルエクスプローラ）

| キー | 操作 |
| --- | --- |
| `-` | 現在のファイルの親ディレクトリを開く |
| `<CR>` | ファイル/ディレクトリを開く |
| `-` (Oil内) | 親ディレクトリへ移動 |
| `g?` | ヘルプ表示 |

ファイル操作（通常のバッファ編集と同じ）:

| 操作 | やり方 |
| --- | --- |
| 新規ファイル | `o` で新行追加 → ファイル名を入力 → `:w` |
| 名前変更 | ファイル名を直接編集 → `:w` |
| 削除 | `dd` で行削除 → `:w` |
| 移動 | `dd` → 移動先で `p` → `:w` |

## LSP（IDE 機能）

| キー | 操作 |
| --- | --- |
| `gd` | 定義へジャンプ |
| `gr` | 参照一覧 |
| `K` | ホバー情報表示 |
| `<Space>rn` | シンボル名前変更 |
| `<Space>ca` | コードアクション |

対応言語: Ruby (`ruby_lsp`)

## Gitsigns（Git 差分表示）

サインカラム（行番号の左）に自動表示:

| 記号 | 意味 |
| --- | --- |
| 緑の縦線 | 追加行 |
| 青の縦線 | 変更行 |
| 赤の三角 | 削除行 |

## Treesitter（シンタックスハイライト）

自動で有効。手動操作は不要。

| コマンド | 操作 |
| --- | --- |
| `:TSUpdate` | パーサーを更新 |
| `:TSInstall {言語}` | 新しい言語パーサーを追加 |

対応言語: lua, ruby, javascript, typescript, json, yaml, html, css, bash, markdown, vim, vimdoc

## lazy.nvim（プラグイン管理）

| コマンド | 操作 |
| --- | --- |
| `:Lazy` | プラグイン管理画面を開く |
| `:Lazy sync` | プラグインの更新・インストール |
| `:Lazy health` | 状態チェック |

## ウィンドウ操作（グローバル）

| キー | 操作 |
| --- | --- |
| `<C-h>` | 左のウィンドウへ移動 |
| `<C-j>` | 下のウィンドウへ移動 |
| `<C-k>` | 上のウィンドウへ移動 |
| `<C-l>` | 右のウィンドウへ移動 |
| `<Esc>` | 検索ハイライト解除 |
